# Kleiderspenden-Portal

Eine einfache Webanwendung zur Registrierung von Kleiderspenden im Rahmen der Fallstudie IPWA01-01.

## Funktionen

- Auswahl zwischen Übergabe an der Geschäftsstelle und Abholung durch Sammelfahrzeug  
- Erfassung von Art der Kleidung und Krisengebiet  
- Validierung von Abholadressen (PLZ-Bereiche für Wien)  
- Bestätigungsseite mit Datum, Uhrzeit und Zusammenfassung  
- Responsives Layout (Desktop, Tablet, Smartphone)  
- Modernes UI-Design (Kartenlayout, Navigation, Buttons, Animationen)

## Technologien

- HTML  
- CSS  
- JavaScript (Vanilla JS, ohne Frameworks)

## Projektstruktur

index.html  
ueber-uns.html  
kontakt.html  
rechtliches.html  
style.css  
script.js

## Zweck

Dieses Projekt dient ausschließlich der Erstellung einer Fallstudie für den Studiengang der IU Internationale Hochschule.
